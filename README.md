# CNVG Front

Next.js проект с системой компонентов и дизайн-токенов.

## Технологии

- Next.js 16.1.0
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- Phosphor Icons

## Компоненты

### UI компоненты

#### Button
Кнопка с поддержкой различных вариантов и размеров.

- **Размеры**: `small`, `medium`, `big`
- **Варианты**: `primary`, `card`, `cardAccent`, `secondary`, `secondaryAccent`, `secondaryOnColor`
- **Типы**: `text`, `leftIcon`, `rightIcon`, `icon`
- **Цвета акцента**: `green`, `red`, `orange`, `yellow`, `cyan`, `blue`, `indigo`, `purple`, `pink`
- Поддержка ссылок через `url` или обработчика через `onClick`

#### Icon
Компонент иконок на базе Phosphor Icons.

- **Варианты**: `big` (40x40, градиент), `small` (24x24, градиент), `ui16` (16x16), `ui12` (12x12)
- Поддержка кастомных градиентов (угол, цвета)
- Кэширование компонентов иконок

#### CardLink
Карточка-ссылка с иконкой, заголовком и описанием.

- Иконка с градиентом
- Заголовок и описание
- Индикатор перехода (caret)
- Hover эффекты

#### CardSimple
Простая карточка для контента.

- Опциональные заголовок, описание, иконка
- Поддержка произвольного контента через `children`
- Верхняя граница с кастомным цветом

### Layout компоненты

#### Section
Секция страницы с заголовком и описанием.

- Опциональные заголовок и описание
- Поддержка ссылки в заголовке
- Контейнер с максимальной шириной

#### SectionHero
Герой-секция страницы.

- Хлебные крошки
- Заголовок и описание
- Опциональная кнопка действия (адаптивная: иконка на мобильных, текст + иконка на десктопе)

#### SectionBreadcrumbs
Навигационные хлебные крошки.

- Горизонтальная прокрутка при переполнении
- Обрезка длинных меток (макс. 20 символов)
- Поддержка ссылок и статичных элементов

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
